# Расширение AIAssistant для 1С:Предприятие

HTTP-сервис для выполнения запросов к витринным регистрам из Python API.

Итоговый endpoint: `POST http://<1c-server>/<base>/hs/ai/query`

---

## 1. Создание расширения

Конфигуратор → Конфигурация → Расширения конфигурации → Добавить:

| Параметр | Значение |
|----------|----------|
| Имя | AIAssistant |
| Префикс | aihlp_ |
| Назначение | Дополнение |

---

## 2. HTTP-сервис QueryService

В расширении: правой кнопкой → Добавить → HTTP-сервисы → HTTP-сервис.

| Параметр | Значение |
|----------|----------|
| Имя | QueryService |
| Корневой URL | ai |

### 2.1 Шаблон URL

В QueryService → Шаблоны URL → Добавить:

| Параметр | Значение |
|----------|----------|
| Шаблон | /query |

### 2.2 Метод

В шаблоне /query → Методы → Добавить:

| Параметр | Значение |
|----------|----------|
| Метод HTTP | POST |
| Имя обработчика | ОбработкаВызова |

### 2.3 Код модуля HTTP-сервиса

В модуле QueryService вставить:

```bsl
Функция ОбработкаВызова(Запрос)
    Попытка
        ТелоJSON = Запрос.ПолучитьТелоКакСтроку();
        ЧтениеJSON = Новый ЧтениеJSON;
        ЧтениеJSON.УстановитьСтроку(ТелоJSON);
        Данные = ПрочитатьJSON(ЧтениеJSON);

        ТекстЗапроса = Данные.query;
        Параметры = Данные.params;

        // Проверка: только ВЫБРАТЬ
        Если НЕ СтрНачинаетсяС(ВРег(СокрЛП(ТекстЗапроса)), "ВЫБРАТЬ") Тогда
            Возврат СформироватьОтвет(Ложь, , "Разрешены только запросы ВЫБРАТЬ");
        КонецЕсли;

        Запрос1С = Новый Запрос;
        Запрос1С.Текст = ТекстЗапроса;

        // Установить параметры
        Для Каждого КлючЗначение Из Параметры Цикл
            Значение = КлючЗначение.Значение;
            // Если строка похожа на дату — преобразовать
            Если СтрДлина(Значение) = 10 И СтрНайти(Значение, "-") > 0 Тогда
                Значение = Дата(СтрЗаменить(Значение, "-", "") + "000000");
            КонецЕсли;
            Запрос1С.УстановитьПараметр(КлючЗначение.Ключ, Значение);
        КонецЦикла;

        РезультатЗапроса = Запрос1С.Выполнить();
        Таблица = РезультатЗапроса.Выгрузить();

        // Конвертация в массив объектов
        Данные = Новый Массив;
        Для Каждого Строка Из Таблица Цикл
            Объект = Новый Структура;
            Для Каждого Колонка Из Таблица.Колонки Цикл
                Объект.Вставить(Колонка.Имя, Строка[Колонка.Имя]);
            КонецЦикла;
            Данные.Добавить(Объект);
        КонецЦикла;

        Усечено = Данные.Количество() >= 1000;
        Возврат СформироватьОтвет(Истина, Данные, , Данные.Количество(), Усечено);

    Исключение
        Возврат СформироватьОтвет(Ложь, , ОписаниеОшибки());
    КонецПопытки;
КонецФункции

Функция СформироватьОтвет(Успех, Данные = Неопределено,
    Ошибка = Неопределено, Всего = 0, Усечено = Ложь)

    Ответ = Новый Структура;
    Ответ.Вставить("success", Успех);
    Ответ.Вставить("data", ?(Данные = Неопределено, Новый Массив, Данные));
    Ответ.Вставить("total", Всего);
    Ответ.Вставить("truncated", Усечено);
    Ответ.Вставить("error", Ошибка);

    ЗаписьJSON = Новый ЗаписьJSON;
    ЗаписьJSON.УстановитьСтроку();
    ЗаписатьJSON(ЗаписьJSON, Ответ);

    HTTPОтвет = Новый HTTPСервисОтвет(200);
    HTTPОтвет.УстановитьТелоИзСтроки(ЗаписьJSON.Закрыть());
    HTTPОтвет.Заголовки.Вставить("Content-Type", "application/json; charset=utf-8");
    Возврат HTTPОтвет;
КонецФункции
```

## 3. Роль ТолькоЧтениеВитрин

В расширении: правой кнопкой → Добавить → Роли → Роль.

| Параметр | Значение |
|----------|----------|
| Имя | ТолькоЧтениеВитрин |

### Права роли

Установить право **Чтение** на витринные регистры накопления, используемые в дашбордах. Конкретный список регистров зависит от конфигурации 1С Аналитики.

Также установить:
- **Использование** на HTTP-сервис `QueryService`
- Снять все остальные права (запись, изменение, удаление, добавление)

---

## 4. Пользователь ai_assistant

Администрирование → Пользователи → Добавить:

| Параметр | Значение |
|----------|----------|
| Имя | ai_assistant |
| Роль | ТолькоЧтениеВитрин |
| Аутентификация | Базовая (HTTP Basic Auth) |

Задать пароль. Сохранить в `.env` на GPU-сервере:

```env
ONEC_USER=ai_assistant
ONEC_PASSWORD=<пароль>
```

## 5. Публикация базы

Если база ещё не опубликована через веб-сервер:

### Apache (Linux)

1. Установить модуль публикации 1С:
   ```bash
   sudo apt install 1c-enterprise-*-ws
   ```

2. Опубликовать базу через конфигуратор:
   Администрирование → Публикация на веб-сервере:
   - Имя публикации: `base` (или имя вашей базы)
   - Отметить HTTP-сервисы → QueryService

3. Перезапустить Apache:
   ```bash
   sudo systemctl restart apache2
   ```

Если база уже опубликована — убедиться, что в файле публикации (`default.vrd`) присутствует:

```xml
<httpServices publishByDefault="true"/>
```

Или добавить конкретный сервис:
```xml
<httpService name="QueryService" rootUrl="ai"/>
```

## 6. Проверка

```bash
curl -u ai_assistant:<пароль> \
  -X POST \
  -H "Content-Type: application/json" \
  http://<1c-server>/<base>/hs/ai/query \
  -d '{"query": "ВЫБРАТЬ ПЕРВЫЕ 1 1 КАК Тест", "params": {}}'
```

Ожидаемый ответ:

```json
{
  "success": true,
  "data": [{"Тест": 1}],
  "total": 1,
  "truncated": false,
  "error": null
}
```

## Формат запроса / ответа

### Запрос (POST JSON)

```json
{
  "query": "ВЫБРАТЬ ПЕРВЫЕ 1000 Период, СУММА(Сумма) КАК Сумма ИЗ РегистрНакопления.ВитринаВыручка.Обороты(&Начало, &Конец, , , ) СГРУППИРОВАТЬ ПО Период",
  "params": {
    "Начало": "2025-01-01",
    "Конец": "2025-03-31"
  }
}
```

### Ответ (JSON)

```json
{
  "success": true,
  "data": [
    {"Период": "2025-01-01T00:00:00", "Сумма": 4100000},
    {"Период": "2025-02-01T00:00:00", "Сумма": 3800000},
    {"Период": "2025-03-01T00:00:00", "Сумма": 4500000}
  ],
  "total": 3,
  "truncated": false,
  "error": null
}
```
